# 🚀 どのPCでも一発起動！固定URL公開対応 Web通知システム

登録不要・完全無料・固定URLで、**お客様がご自宅からでも**いつでもお知らせを見られる通知システムです。

---

## ⚙️ 最初にやること（1回だけ）

**`server.py`** をテキストエディタで開き、お店の名前を設定します。

```python
# server.py の 32行目あたり
STORE_NAME = "otemise"   # ← ここを半角英数のお店の名前に変更
```

例: `yamadashoten` にすると `https://yamadashoten.serveo.net` が固定URLになります。

---

## 💻 毎日の起動方法（ダブルクリックするだけ！）

| OS | 起動ファイル | 操作 |
|:---|:---|:---|
| **Mac** | `start.command` | ダブルクリック |
| **Windows** | `start.bat` | ダブルクリック |

起動すると以下が自動で行われます：
1. ✅ ローカルサーバーが立ち上がる
2. ✅ インターネット公開（serveo.net SSHトンネル）が開通する
3. ✅ ブラウザが自動で開く

---

## 📱 お客様への案内URL

起動後、画面に以下のURLが表示されます。

```
📱 お客様用（自宅からもOK）: https://お店名.serveo.net
⚙️  管理者用（自宅からもOK）: https://お店名.serveo.net/admin.html
```

このURLを **QRコードにしてチラシ・ポスターに掲示するだけ**でお客様がどこからでもアクセスできます。

---

## ❗ 重要なポイント

- **PCの電源を入れてこの画面（ターミナル）が開いている間だけ**、お客様がアクセスできます。
- 毎日お店を開けたときにダブルクリックで起動し、閉めたときに Ctrl+C で停止するのがおすすめです。
- URLは毎回同じ固定URLになります（`STORE_NAME`が同じであれば）。

---

## 🌐 仕組みについて

```
お客様のスマホ
    ↓（https://お店名.serveo.net にアクセス）
serveo.net（無料・登録不要のSSHトンネルサービス）
    ↓（店舗PCへ転送）
店舗PCのPythonサーバー（localhost:8000）
    ↓
SQLiteデータベース（メッセージ・購読情報を保存）
```

---

## 🛠️ 管理者パスワードの変更

起動コマンドの前に環境変数を設定します（デフォルト: `admin123`）。

**Mac/Linux:**
```bash
ADMIN_PASSWORD=新しいパスワード python3 server.py
```

**Windows:**
```cmd
set ADMIN_PASSWORD=新しいパスワード && python server.py
```
